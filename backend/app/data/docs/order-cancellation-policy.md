# Order Cancellation Policy

- **Document Title:** Order Cancellation Policy
- **Document Type:** Customer-Facing Policy
- **Intended Audience:** Customers
- **Last Updated:** March 12, 2026

## Overview

Northbeam Goods allows cancellations only before an order reaches a shipment-ready stage. Because orders move quickly through fulfillment, cancellation requests are not guaranteed.

## When Cancellation Is Allowed

Cancellation is generally possible when the order is still in one of these statuses:

- **Pending Payment**
- **Confirmed**
- **Processing**

If cancellation is approved before shipment, the order will be canceled and a refund will be initiated to the original payment method.

## When Cancellation Is No Longer Possible

Cancellation is usually not possible when the order has reached:

- **Packed**
- **Shipped**
- **Out for Delivery**
- **Delivered**

At that stage, the customer should wait for delivery and request a return if the item is eligible under the Returns and Replacements Policy.

## High-Risk or Review Orders

Orders under payment review, address verification, or fraud screening may not be canceled until the review is complete. Support should not promise cancellation during manual review.

## Customer Support Guidance

### If Cancellation Is Still Possible
Support should confirm that:

- cancellation has been requested
- the request is being checked against the current order status
- refund timing depends on payment provider processing

**Suggested response:**  
“Your cancellation request has been submitted. If the order has not moved to Packed or Shipped status, we will cancel it and issue a refund to your original payment method.”

### If Cancellation Is No Longer Possible
Support should explain:

- the order has already moved too far in fulfillment
- cancellation is no longer available
- return options may be available after delivery

**Suggested response:**  
“Your order has already moved to Packed or Shipped status, so we can no longer cancel it. Once it arrives, you may request a return if the item qualifies under our return policy.”

### If Status Is Unclear
Support should not guess.

**Suggested response:**  
“I’m checking the latest fulfillment status now. I’ll confirm whether cancellation is still possible before I give you next steps.”

## Refund Timing for Cancellations

For approved cancellations:

- refund initiation typically occurs within **3 business days**
- bank or card posting usually takes **5 to 10 business days**

## Exceptions

Northbeam Goods may refuse cancellation when:

- the package has already been transferred to the carrier
- the order includes custom or made-to-order items already in production
- the request is linked to suspected fraud or payment dispute handling
